# Research Grants References

This folder contains supporting guidance for the `research-grants` skill.

## Files

- `nsf_guidelines.md`: NSF proposal structure, review criteria, and broader impacts expectations
- `nih_guidelines.md`: NIH specific aims, significance, innovation, and approach guidance
- `doe_guidelines.md`: DOE proposal positioning, office alignment, and project execution guidance
- `darpa_guidelines.md`: DARPA framing for program fit, ambitious milestones, and transition plans
- `nstc_guidelines.md`: NSTC proposal conventions and bilingual planning notes
- `specific_aims_guide.md`: How to draft and stress-test a one-page aims document
- `broader_impacts.md`: Practical strategies for stronger NSF-style broader impacts sections

## How to Use

1. Start with the agency-specific guideline for the target solicitation.
2. Use `specific_aims_guide.md` when shaping the research narrative.
3. Use `broader_impacts.md` when the proposal needs societal, educational, or workforce impact planning.
4. Cross-check the solicitation itself before final submission because agency and program rules change.
